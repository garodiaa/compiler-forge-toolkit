Supported Grammars

  • A -> A c | A a d | b d | ϵ
    (All tokens must be separated by space characters)
  • A -> A c
      | A a d
      | b d
      | ϵ
  • S -> A a | b
    A -> A c | S d | ϵ
  • (Copy ϵ to input if needed)

Example

  • S -> S S + | S S * | a
  • S -> 0 S 1 | 0 1
  • S -> ( L ) | a
    L -> L , S | S
  • S -> a S b S | b S a S | ϵ
  • S -> S + S | S S | ( S ) | S * | a
  • S -> + S S | * S S | a
